Desafio Git

O desafio é muito "simples". Faça um fork desse projeto para para a sua conta mo github, depois baixe ele via git clone para o seu computador. Adicione seu nome na lista que se encontra abaixo nessa página. Commit suas modificações, faça um push para o fork na sua conta. Depois submeta um pull request para o repositório original desse projeto.

Esse é o fluxo para poder trabalhar de forma colaborativa no github. Todos são livres para participar. Quem não puder não tem problema. Uma dica importante é: antes de fazer qualquer coisa no github tente gerar sua chave ssh pública e adicione na sua conta.
